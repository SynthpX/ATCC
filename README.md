### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/SynthpX/ATCC.git
   ```
2. Install requirements.txt
   ```sh
   pip install -r requirements.txt
   ```
3. Run app.py
   ```sh
   python app.py
   ```
